Quick Start

Gmvault can be installed on Windows, Linux and Mac Os X.
Follow the guide to install on the platform of your choice.

Windows Quick Start

Windows Installation

Go to the Download Page to download the package for the platform of your choice. On Windows, Gmvault can be installed using an installer called gmvault_setup.exe.
Download it and double click on it to launch the installation.

  • 1. Launch the gmvault_setup.exe windows installer.
  • 2. Accept the software licensing prior to install it on your PC.


  • 3. Choose where to install Gmvault.
  • 4. Agree on the creation of a Desktop shortcut for Gmvault (recommended).

The gmvault installer should have installed the gmvault exe and a gmvault-shell from where you can easily use gmvault.
Gmvault-shell can be launched via the desktop shortcut.
